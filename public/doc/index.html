This project is a starting point and a template for building PickMe service APIs using PHP.

/auth

post /auth/login

post Login to the device and get a token.

post /auth/login

Login to the device and get a token.

  • Request
  • Response

Body

Type: application/json

{
                                    	"phone": 111111111,
                                    	"password":"password"
                                    }

Responses

  • 200: Successfully logged-in
  • 422: Validation error
  • 500: An exception occurred while logging in

post /auth/logout

post Logout from the device.

post /auth/logout

Logout from the device.

  • Request
  • Response

Responses

  • 204: Successfully logged-out
  • 500: An exception occurred while logging out
